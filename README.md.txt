# 🍔 Food Delivery Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze food delivery operations and provide business insights.

The dashboard helps users monitor delivery performance, analyze order trends, and compare orders across different city categories.

---

## 🚀 Dashboard Preview

![Dashboard](page1.png)

---

## 📊 Features

- Average Delivery Time KPI
- Total Orders KPI
- Delivery Time Trend by Date
- Orders by City
- Interactive City Slicer
- Detailed Order Table
- Cross-filtering between visuals

---

## 🛠 Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- CSV Dataset
